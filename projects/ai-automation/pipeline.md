# Pipeline

_Leads + clients tracker. Update after each outreach session._

## Stages
1. **Prospect** — identified but not contacted
2. **Outreach** — first message sent
3. **Conversation** — replied, discussing fit
4. **Proposal** — sent offer/quote
5. **Won** — signed, paid
6. **Lost** — no/ghosted (log why)
7. **Delivered** — project shipped, post-mortem done
