# Synthesis — Discovery Insights

_Living document. Обновляется после каждого интервью (начиная с 5-го). Цель: увидеть паттерны, которые повторяются независимо у разных людей._

## Метод

После каждого интервью:
1. Открыть свежие заметки из `interviews/`
2. Выписать упомянутые боли в список ниже
3. Если боль повторилась (≥2 риелтора назвали независимо) — поднять в `## Confirmed patterns`
4. Если ≥3 подтверждений — это кандидат на дизайн первого Volat'а

## Golden Rule

**Не проектируй Volat пока одна боль не подтвердится 3+ раз независимо.** Одно интервью = анекдот. Три — паттерн.

## Decision point: ready to design offer?

Checklist:
- [ ] 10+ интервью проведено
- [ ] Есть 2-3 подтверждённых паттерна боли (confirmed)
- [ ] Минимум 3 человека сказали "я бы заплатил"
- [ ] Понятна экономика (чек, частота, LTV-потенциал)
- [ ] Volat-гипотеза под одну из болей звучит как "да, это решит"

Когда все галочки — переходим в `offer.md`.
