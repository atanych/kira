# Research — Volat AI Discovery

_Дискавери-фаза для V1 вертикали: RU-speaking риелторы диаспоры (Dubai/Cyprus/Georgia/Turkey)._

## Цель
Провести 10-15 discovery-звонков с реальными риелторами **до** дизайна оффера. Найти 2-3 настоящие боли (не гипотезы), под которые мы потом спроектируем первого Volat'а.

## Принцип
**Discovery-first.** Никаких питчей, никаких "вот наш продукт", никаких цен на этом этапе. Только слушаем, задаём вопросы по The Mom Test, собираем патте­рны.

## Файлы

- [[framework]] — Mom Test-вопросы, listening guide, red flags
- [[outreach-template]] — no-pitch DM шаблон для discovery
- [[target-icp]] — критерии ICP + где искать диаспорных риелторов
- [[targets]] — трекер 20+ потенциальных интервью
- [[interviews/_template]] — шаблон заметок по интервью
- [[synthesis]] — living document: паттерны и инсайты накапливаются сюда

## Пайплайн

```
1. ICP критерии    → target-icp.md
2. Сбор 20 targets → targets.md
3. Outreach DM     → outreach-template.md
4. Звонок          → interviews/YYYY-MM-DD-name.md (скопировать _template.md)
5. Синтез после 5+ → synthesis.md (обновляется после каждого интервью)
6. Re-evaluate     → после 10-15 интервью → решение по оффёру
```

## Критерии выхода из discovery-фазы

Готовы дизайнить оффер когда:
- [ ] 10+ интервью проведено
- [ ] 3+ паттерна боли повторяются независимо (≥3 риелтора назвали)
- [ ] Понятна экономика: сколько стоит лид, cycle time, конверсия
- [ ] Есть 2-3 человека сказавших "если бы вы это сделали, я бы заплатил"
