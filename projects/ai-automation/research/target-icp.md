# Target ICP — Discovery Phase

_Кого ищем для 10-15 discovery-звонков. Это ICP для ИНТЕРВЬЮ, не для продажи (пока)._

## Primary ICP (фокус 80% усилий)

**Кто:**
- Русскоязычный риелтор (РФ/BY/UA/KZ диаспора)
- Работает в одной из: **Дубай, Кипр, Грузия, Турция** (основная локация — там)
- 2+ года опыта в недвижимости на этом рынке
- Работает соло или в маленькой команде (1-5 человек)
- Активен в Instagram / Telegram (есть профессиональный профиль)
- Закрывает минимум 1-2 сделки в месяц
- Клиенты: 70%+ из РФ/СНГ

**Specialization (любая из):**
- Продажа новостроек застройщиков
- Ресейл жилого
- Инвестиционная недвижимость (commercial / flipping)
- Релокация/ВНЖ через недвижку

**Почему они:**
- Есть реальная операционная нагрузка (лиды / кросс-таймзоны / переговоры)
- Достаточно зарабатывают чтобы платить за инструменты ($500-2000/mo — нормально)
- Технологически открыты (живут в Instagram/TG, используют CRM)
- Диаспора = знакомый RU-контент, плюс повод для YouTube

## Secondary ICP (если primary не хватает)

- Брокеры в агентствах среднего размера (5-20 человек) — руководители
- Риелторы в Бали / Таиланде работающие с RU-клиентом
- Consultants по релокации в EU (Португалия, Испания)

## Anti-ICP — НЕ берём на звонок

❌ Риелторы без опыта (< 1 года)
❌ Риелторы подрабатывающие (основная работа другая)
❌ Крупные агентства (20+ агентов) — там корпоративные процессы, не наш кейс
❌ Специалисты только по Москве/СПб — не диаспора
❌ Те кто в последний год сменил 3+ ниши — не коммит
❌ Инфобизнесмены "риелтор-коуч" (учат а не продают)

## Где искать

### Instagram (главный источник)

**Поиск по тегам:**
- #недвижимостьдубай #недвижимостькипр #недвижимостьгрузия #недвижимостьтурция
- #риелтордубай #риелторкипр #риелторгрузия
- #дубайнедвижимость #риелторпокипру
- #инвестициивдубае #dubairealestate (русскоязычные профили в комментах)

**Поиск по локации:**
- Geotag Dubai / Limassol / Tbilisi / Istanbul + русский язык профиля

**Через хэштеги застройщиков:**
- #emaar #damac #aldar → русскоязычные риелторы которые их продают

### Telegram

- Каналы "Недвижимость Дубай RU" / "Кипр инвест" — смотреть авторов и комментаторов
- Чаты диаспоры (expat chats) — там риелторы часто светятся

### YouTube

- Поиск "купить квартиру в Дубае" / "инвестиции в недвижимость Грузия"
- Русскоязычные каналы риелторов-блогеров (10-50k подписчиков — sweet spot)

### LinkedIn

- Job title: "Real Estate Agent" / "Property Consultant" + локация
- Search language: Russian posts

### Рефералы (после первых 2-3 звонков)

- Каждый звонок заканчивать "кого порекомендуете поговорить"
- Это даёт самые качественные лиды

## Фильтр перед outreach

Перед тем как отправить DM, проверяем 30 секунд:
1. Есть ли реальные посты про недвижку (не только лайфстайл)?
2. Есть ли контакт / сайт / канал (не только айфон-фото)?
3. Последний пост ≤ 2 месяца назад?
4. Подписчиков: 500-20k (микро-средний, не знаменитости)?

Если все 4 — ок, добавляем в `targets.md`.

## Geo-распределение (чтобы не перекосить выборку)

Цель 10-15 интервью:
- Dubai: 5-6 (самый большой рынок, больше всего риелторов)
- Cyprus: 3-4
- Georgia: 2-3
- Turkey: 2-3
- Other (Бали/EU): 1-2

Если один рынок вылезет в топ по болям — ок, сфокусируемся.
Если все одинаковые — значит Volat можно делать cross-market.
