# Outreach Templates — Discovery DM

_Цель: получить 45-минутный разговор. НЕ продавать. НЕ питчить Volat AI._

## Принципы

1. **Не упоминаем продукт.** Мы пока ничего не продаём.
2. **Короткое сообщение.** Длинные DM не читают.
3. **Честная причина.** "Изучаю рынок" — правда на этом этапе.
4. **Никаких форм / Calendly в первом сообщении.** Сначала "да, ок", потом ссылка.
5. **Не флексим экспертизой.** Мы учимся, а не консультируем.

---

## Template A — Instagram DM (основной канал)

```
Привет, [Имя]!

Я [Vladimir], делаю ресёрч по рынку недвижимости для русскоязычных клиентов
в [Dubai/Кипре/Грузии] — как устроена работа риелторов на этом рынке,
какие процессы, что откуда берётся.

Увидел ваш профиль — вы явно в теме. Было бы реально полезно 30-40 минут
поговорить про то как устроена ваша работа. Ничего не продаю, правда
просто собираю картинку рынка.

Можем созвониться на этой или следующей неделе?
```

**Почему работает:**
- Честно говорит кто ты и зачем
- "Ничего не продаю" сразу снимает защиту
- Комплимент экспертизе (но не сопли)
- Конкретный ask (30-40 мин)

## Template B — Telegram (если есть ник)

Более personable чем Instagram:

```
[Имя], привет!

Наткнулся на ваш канал/профиль — работаете с русскоязычными по недвижке
в [city]. Делаю ресёрч рынка, хочу понять как реально устроена работа
риелторов диаспоры (процессы, лидген, боли).

Не продажа — реально интервью для рисёрча. 30-40 минут удобно на этой
неделе?
```

## Reply-шаблоны (после "да, давайте")

### Согласие → просьба выбрать слот

```
Супер, спасибо! Вот мой календарь: [Calendly link]
Выберите любой удобный слот. Если ничего не подходит — просто
предложите 2-3 варианта, подстроюсь.

На звонке ничего не надо готовить — просто буду задавать вопросы
по вашей работе. Запишу для своих заметок, если не против.
```

### Подтверждение за 24 часа до

```
[Имя], завтра в [time] созвон, подтверждаю. Zoom-линк:
[link]

Если что-то изменится — просто напишите, сдвинем.
```

### После звонка (спасибо + ask на реферал)

```
[Имя], спасибо большое за разговор — очень ценные инсайты по
[X конкретный топик из звонка].

Если у вас в кругу есть коллеги-риелторы диаспоры с кем тоже
стоило бы поговорить — буду благодарен за знакомство.

Если в будущем соберу что-то полезное по этому рынку, поделюсь
первой очередью с вами.
```

## Что НЕ писать в DM

❌ "Мы разрабатываем AI-решение для риелторов..."
❌ "У нас есть продукт который поможет вам..."
❌ "Хочу предложить решение для вашей проблемы..."
❌ "Вам нужна автоматизация? У нас есть..."
❌ Длинные истории о себе (3+ предложения биографии)
❌ Calendly-ссылка в первом сообщении
❌ Эмодзи в количестве больше 1

## Метрики

Считаем в `targets.md`:
- Sent / Reply rate / Booked rate / Show rate

Target: из 20 DM → 5-8 ответов → 3-5 звонков (реалистично).
Чтобы набрать 10-15 звонков — нужно 40-60 DM.
