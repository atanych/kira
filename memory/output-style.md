# Output Style

How to present results: show what changed, pick the right format, fact-check on doubt.

> [!tip] Always show what changed
> Don't just say "done" — show what was actually created/changed.

> [!tip] Format choice — PDF vs PNG
> Text-heavy reports and tables → **PDF**, not PNG. Telegram compresses PNG, text becomes blurry. PDF is sent as a document without compression → можно зумить, цифры читаются на телефоне. PNG only for short visuals / charts with minimal text.

> [!tip] Fact-check on user doubt
> When [[Vovan]] questions you ("ты уверен?", "точно нет такого варианта?") — re-check via real fact-finding, don't double down on the first answer. First thought ≠ final answer.
