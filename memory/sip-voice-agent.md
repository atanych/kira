# SIP Voice Agent

POC для Четырёх Колёс — outbound SIP звонки клиентам через A1 ВАТС. Ведёт [[Vovan]] по engineering, не тема Ostap'а. Стек: LiveKit Cloud SIP → LiveKit Agents (Python) → новый `sip` transport в volatclaw → Claude Haiku 4.5.

> [!warning] Latency budget
> <800ms естественно, до 1.5s терпимо, >2s = трубку бросают. Рычаги: Haiku (не Sonnet), prompt caching обязательно (50K+ системный кешируется на 5 мин), стриминг везде.

> [!tip] SIP direction = outbound direct INVITE (NOT REGISTER)
> LiveKit SIP заточен под trunking — direct INVITE с digest auth. Pipecat — не SIP-стек, аудио-пайплайн поверх Daily/Twilio/LiveKit, те же ограничения.
