Дифференцируй: донатерам — MVP/тост по имени; постоянникам без денег — звания по стажу, перекличка, лотерея, голосование (никого не выделять как 'второй сорт').

Альтернатива по премиуму: Godox V1-C +0-100 (круглая голова GN60, литий быстрая зарядка). Скиновое сглаживание на R6 II: нет SCN-режима, есть Smooth Skin Movie для видео; для фото — Q-меню → Picture Style → Портрет → INFO → Резкость -2/-3, Цветовой тон -1/-2. ND-фильтр на резьбу опционально (K&F/NiSi, нужен диаметр объектива).
