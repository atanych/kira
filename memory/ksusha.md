Решение: новый аккаунт с нуля под VPN — удалить TikTok → перезагрузить телефон → VPN Frankfurt-1 connected → проверить IP на whatismyipaddress.com (Germany) → переустановить TikTok → SIM вынуть перед регистрацией (TikTok может читать оператора) → регистрация через email (НЕ Apple/Google login — те подтянут реальную локацию из Apple/Google акка) → 2-3 дня прогрев (листать ленту 20-30 мин/день под VPN без стримов) → проверить регион в Settings → Account → Region = Germany.
