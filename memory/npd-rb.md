# НПД РБ

Профессиональный подоходный налог (Налог на профдоход) в Беларуси — режим для физлиц-самозанятых. Глава 40 НК РБ + Постановление Совмина №457 от 28.06.2024 (Приложение 2 — перечень 87 разрешённых видов).

> [!warning] С 01.07.2026 введён минимальный НПД 45 BYN/мес
> 45 BYN/мес для всех плательщиков, 18 BYN/мес для пенсионеров. Платить даже если дохода в месяце не было. Не платишь 3 раза подряд → автоматически снимают статус НПД (восстановить можно через 6 мес). 24 месяца без активности → снимают автоматически.

## Ставки
- **10%** — физлица + первые 60К BYN/год от юрлиц
- **20%** — юрлица свыше 60К BYN/год
- **4%** — пенсионеры

## Ключевые пункты для IT (Приложение 2)
- **п.16** — удалёнка через интернет для физлиц И организаций
- **п.63** — разработка ПО + тестирование (ОКЭД 6201/63119)
- **п.83** — установка/настройка ПО (ОКЭД 6209)

## Аренда нежилого помещения
- **п.27 Приложения 2 Постановления №457** — физлицо на НПД может сдавать **ТОЛЬКО ОДИН объект ИЛИ ОДНУ ЧАСТЬ одного объекта одновременно**. Больше — обязательно ИП (подоходный 20% с разницы доход-расход).
- **Возмещение коммуналки** арендаторами больше не облагается НПД если прописано ОТДЕЛЬНЫМ пунктом в договоре (не включено в арендную плату) — важный лайфхак для экономии.

- [[2026-07-17]] [[npd-rb]] НПД РБ применимость к продаже AI-ботов / AI-ассистентов. **Скорее да, но зависит от формы:** разовая разработка бота под клиента = чётко подходит по **п.63 разработка ПО (ОКЭД 6201)**. Установка/настройка готового бота у клиента = **п.83 (ОКЭД 6209)**. Удалённые услуги через интернет = **п.16**. **Подписка (SaaS) — спорно** — налоговая может квалифицировать как 'реализация ПО', это уже не НПД. Тактика: **разовые проекты + мелкое сопровождение** → НПД (10% ставка); **много B2B-подписок** → ИП общая система (20% с прибыли, можно счета-фактуры юрлицам); **обороты сотни тысяч BYN/год** → ООО на УСН 6% (солиднее для клиентов). Юрлицам-нерезидентам НПД разрешает, но нужно валютное регулирование через банк. Юрлицо на общей системе часто просит ИП/ООО (физлицо НПД не даёт счёт-фактуру для НДС). Уточнять через письменный запрос на nalog.gov.by → 'Обратная связь' (юр.сила) либо звонок в 189 / +375 17 229-79-79.
