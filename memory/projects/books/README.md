# Books

Persistent collection Vovan's книжных конспектов, генерируемых скиллом [[book-summary]]. Каждая книга — отдельный .md с фронтматтером (название, автор, рейтинги, дата), Obsidian-кросс-ссылками и визуальной карточкой.

## Files

### Saved summaries
- [Continuous Discovery Habits](Continuous%20Discovery%20Habits.md) — Teresa Torres on weekly customer touchpoints + opportunity solution trees
- [Good to Great](Good%20to%20Great.md) — Jim Collins: Level 5 leadership, Hedgehog Concept, flywheel
- [Never Split the Difference](Never%20Split%20the%20Difference.md) — Chris Voss FBI negotiation tactics
- [Rich Dad Poor Dad](Rich%20Dad%20Poor%20Dad.md) — Kiyosaki on assets vs liabilities mindset
- [Secrets of the Millionaire Mind](Secrets%20of%20the%20Millionaire%20Mind.md) — Harv Eker on money blueprint
- [The Mom Test](The%20Mom%20Test.md) — Rob Fitzpatrick: customer interviews that don't lie
- [The Personal MBA](The%20Personal%20MBA.md) — Josh Kaufman: business fundamentals primer
- [Think and Grow Rich](Think%20and%20Grow%20Rich.md) — Napoleon Hill classic
- [Unreasonable Hospitality](Unreasonable%20Hospitality.md) — Will Guidara on hospitality-as-strategy

### Subfolders
- `assets/` — визуальные карточки (PNG, 800px), эмбедятся в .md через `![[Book Title.png]]`
- `voss-playbook/` — deep-dive разборы техник из Never Split the Difference (mirroring, labeling, calibrated questions)

## How to add
Запрос Vovan'у: «сделай summary книги X» → книги-skill сам положит .md сюда + ассет в `assets/`. Не править вручную, если книга есть в коллекции — пере-сгенерить через скилл.
