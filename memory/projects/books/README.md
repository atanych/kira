# Books

Persistent collection Vovan's книжных конспектов, генерируемых скиллом [[book-summary]]. Каждая книга — отдельный .md с фронтматтером (название, автор, рейтинги, дата), Obsidian-кросс-ссылками и визуальной карточкой.

## Files

### Saved summaries
- [[Continuous Discovery Habits]] — Teresa Torres on weekly customer touchpoints + opportunity solution trees
- [[Good to Great]] — Jim Collins: Level 5 leadership, Hedgehog Concept, flywheel
- [[Never Split the Difference]] — Chris Voss FBI negotiation tactics
- [[Rich Dad Poor Dad]] — Kiyosaki on assets vs liabilities mindset
- [[Secrets of the Millionaire Mind]] — Harv Eker on money blueprint
- [[The Mom Test]] — Rob Fitzpatrick: customer interviews that don't lie
- [[The Personal MBA]] — Josh Kaufman: business fundamentals primer
- [[Think and Grow Rich]] — Napoleon Hill classic
- [[Unreasonable Hospitality]] — Will Guidara on hospitality-as-strategy

### Subfolders
- `assets/` — визуальные карточки (PNG, 800px), эмбедятся в .md через `![[Book Title.png]]`
- `voss-playbook/` — deep-dive разборы техник из Never Split the Difference (mirroring, labeling, calibrated questions)

## How to add
Запрос Vovan'у: «сделай summary книги X» → книги-skill сам положит .md сюда + ассет в `assets/`. Не править вручную, если книга есть в коллекции — пере-сгенерить через скилл.
