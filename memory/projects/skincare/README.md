# Skincare

Vovan's skincare routine — комбинированная кожа, лето: жирная T-зона + чёрные точки + старт антиэйдж в 39. Текущий стек, целевая рутина, шоппинг через Золотое Яблоко, журнал.

## Snapshot
- **Тип кожи:** combination (T-зона жирнится летом, остальное нормальная)
- **Возраст:** 39
- **Основные проблемы:** жирность лба/носа летом, чёрные точки на носу, старт антиэйдж
- **Где покупаем:** Золотое Яблоко (goldapple.ru) → Viber-посредник в РБ
- **Бюджет на upgrade:** ~$100 (приоритет — эффективность)
