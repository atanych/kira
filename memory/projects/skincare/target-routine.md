# Target Routine (лето 2026 — упрощённая версия)

## Принципы
- Тёплая вода для умывания (не горячая, не ледяная), прохладная в финале
- 2 умывания в день (утром + вечером)
- **Без чередований через день** — рутина одна и та же утром и вечером
- Никаких BHA / ретинолов пока — начинаем с мягкого старта (niacinamide)
