# Current Routine (на 2026-06-23)

## Утро
1. Умывание **холодной водой** ⚠️ (надо менять — sebum за ночь водой не смывается)
2. **BABE Hydro 24h Light SPF 20** — испанский аптечный, для combination skin, текстура лёгкая. SPF 20 — слабо для лета.

## Вечер
1. Умывание **COSRX Low pH Good Morning Gel Cleanser** — низкий pH ~5-6, мягкий, есть капля BHA + tea tree. Норм базовый клинзер.
2. **Bielenda Skin Clinic Professional Super Power Mezo Cream** — польский, гиалуронка + пептиды. Для combination skin летом ТЯЖЕЛОВАТ — путь к ещё большему блеску. Оставить до осени/зимы.

## Проблемы текущей рутины
- ❌ Утром нет клинзера — водой жир не растворяется
- ❌ SPF 20 — недостаточно для лета (нужно 30+, лучше 50, особенно с acids)
- ❌ Вечерний крем тяжёлый — забивает поры, провоцирует жирность
- ❌ Нет активов: ни BHA для пор, ни ретинола для антиэйдж, ни Vit C для тона
- ❌ Холодная вода для умывания — клинзер плохо пенится, sebum не растворяет
