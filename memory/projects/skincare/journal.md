## Что отслеживаем
- **T-зона блеск** — субъективно по 5-балльной шкале (1 — матовая, 5 — зеркало)
- **Чёрные точки на носу** — есть / уменьшились / почти нет
- **Раздражение** — от ретинола/BHA: жжение, шелушение, краснота
- **Фото нос/лоб** — раз в 2 недели для сравнения
