# Car Investment

Бизнес-проект: пригон EV из Китая под перепродажу в РБ.

**Не путать с** `projects/cars/` — там личная замена XC90 для Vovan. Здесь — про деньги.

## Статус
Pre-launch. Изучаем рынок, считаем формулу, прозваниваем поставщиков.

## Логика сделки
1. Находим EV на китайском сайте (autohome.com.cn / dongchedi / etc.)
2. Считаем landed cost по формуле (см. [[pricing]])
3. Сверяем с рыночной ценой в РБ
4. Если маржа есть — заказываем у поставщика, ждём доставку
5. Продаём в РБ через av.by / Onliner / дилерскую сеть

## Hard constraints
- **Только BEV** (под льготу на ввоз 2026, без растаможки в обычном смысле)
- **Дедлайн льготы:** квота физлиц ~3 673 машин на 7 мая 2026, кончится к октябрю/ноябрю
- **180-day rule:** машина зарегистрированная в Китае <180 дней назад требует After-Sales Service Confirmation Letter. BYD/Denza/Bao не выдают параллельным импортёрам → серые поставки только Avatr/Zeekr/Xpeng/Leapmotor/etc, либо старше 180 дней, либо International Edition
