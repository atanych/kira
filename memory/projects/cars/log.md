- [[2026-06-23]] [[cars]] BYD мастер-аккаунт (2026-06-22 — критично для китайских EV в серой схеме). В ДКП Биулу п.2.1.3 'мастер аккаунт' указан одним словом в списке передаваемых принадлежностей. Без процедуры передачи это пустая строчка — дилер сохраняет полный удалённый контроль через BYD App (геолокация, удалённый запуск, климат, разблокировка, OTA). Что обязательно дописать в АКТ приёма-передачи: (1) email/телефон привязки мастер-аккаунта передаются Покупателю; (2) Продавец гарантирует отвязку всех своих аккаунтов от VIN; (3) при передаче проверяется через приложение BYD, что Покупатель видит авто и имеет права администратора. Без этих 3 пунктов — это фикция.
- [[2026-06-23]] [[cars]] Chinese Apple ID для BYD App (2026-06-22): машина из Китая → нужна **китайская версия BYD App** (比亚迪), доступна только в Mainland China App Store. Гайд: НЕ менять регион основного Apple ID (потеряешь подписки + Apple Developer enrollment), создать второй ID через appleid.apple.com в Safari → регион **'Китай континентальный'** (это и есть Mainland China, не Гонконг/Тайвань/Макао) → виртуальный +86 номер через sms-activate.org или 5sim.net (~$0.5-1) → способ оплаты **None / 无** (Китай разрешает без карты, в отличие от других регионов) → китайский адрес можно нагуглить генератор. Потом на iPhone/iPad: Settings → Apple ID → Media & Purchases → Sign Out (ТОЛЬКО App Store, не iCloud!) → Sign In китайским ID → искать 比亚迪. Android проще: APK с apkpure.com. Интерфейс китайский — переводить камерой Google Translate.

- [[2026-06-25]] [[cars]] Регистрация нового авто в РБ — онлайн-техосмотр (2026-06-24, от Ksusha при постановке BYD на учёт). Новые ТС физически на станцию Белтехосмотра гнать НЕ надо — освобождены от диагностики на 2 года. Нужно только **разрешение на допуск ТС к участию в дорожном движении** — оформляется онлайн через **e-pasluga.by** (Е-Паслуга, портал электронных услуг). Услуга **код 200.15.17**. Логин через МСИ (межбанковская система идентификации, заходит через свой банк). Данные машины подтягиваются из ГАИ автоматически после постановки на учёт. Оплата госпошлины ЕРИП'ом прямо в процессе. Бумажку получать не обязательно — ГАИ видит разрешение в системе. Срок разрешения для нового авто — 2 года. Для электриков на 2026 льгота по госпошлине частично продлена: если на оплате 0 BYN — освобождён, если сумма (~50 BYN) — льготу не продлили на этот сбор. Альтернатива e-pasluga — очно на любой станции Белтехосмотра.
