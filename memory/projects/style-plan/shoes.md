# Обувь

**Жёсткое правило:** никаких беговых / баскетбольных кроссовок с рубашкой. Никаких Air Max, Asics, ярких подошв.

## 3 пары
1. 🥇 Белые минималистичные кеды — универсал. Adidas Stan Smith / Lacoste Carnaby / ECCO Soft / GEOX (200–400 BYN)
2. 🥈 Чёрные / тёмные кожаные кеды — Clarks, ECCO
3. 🥉 (Позже) лоферы или дезерты замшевые бежевые

## Запросы на WB
- `кеды мужские кожаные белые минималистичные`
- `кроссовки мужские кожаные черные классические`

**Если только одна пара для старта — белые кеды.**
