# Roadmap

Чек-лист покупок по этапам. Главный progress tracker — отмечать `[x]` по мере покупок.

## Этап 1: База (минимум для старта стиля)
- [ ] Тёмно-синяя рубашка приталенная
- [ ] Белая рубашка приталенная
- [ ] Бежевые чиносы
- [ ] Тёмно-синие чиносы или тёмные джинсы slim
- [ ] Белые минималистичные кеды

## Этап 2: Расширение
- [ ] Графитовая / тёмно-серая рубашка
- [ ] Голубая рубашка
- [ ] Чёрные кожаные кеды
- [ ] Тонкий пуловер тёмно-синий

## Этап 3: Верх
- [ ] Кожаная куртка чёрная короткая
- [ ] Шерстяное пальто графитовое / тёмно-синее

## Этап 4: Уровень выше
- [ ] Чёрная рубашка
- [ ] Замшевые дезерты / лоферы бежевые
- [ ] Часы с крупным циферблатом (40 мм+)
- [ ] (Опционально) полосатый блейзер летний
