# Стратегия покупок

1. Сначала ножками в **Galileo / Dana Mall** — померить Mango Man slim fit, понять размер вживую
2. Потом **WB** — заказать muscle fit рубашки 2–3 разных бренда в одном размере, выбрать лучшую посадку, остальные вернуть
3. Чиносы тоже **сначала примерить** в магазине — посадка по бёдрам критична
4. Не покупать всё сразу. Этапами. Проверять как сидит, как ощущается.
