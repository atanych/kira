# Стилевой план — Vovan

Long-term переход от спортивного стиля (худи, спортивки) к smart casual. Покупки по этапам, не за неделю.

**Референс:** Tomas Kralow / похожий типаж — Дубай-вайб, slim/muscle fit, тёмные цвета, базовая палитра.

## 📐 Параметры
- Рост: 178 см
- Вес: 84 кг
- Размер: L (но в европейских slim fit может быть XL в плечах)
- Воротник: 41–42
- Тип фигуры: лысый + борода + накачанный, поджарый (не "мясной танк")
- Татуировка на правом бицепсе (можно скрывать/показывать)

## 🎨 Палитра

✅ **Работают:**
- Тёмно-синий (киллер-цвет под светлые глаза)
- Графит / тёмно-серый
- Бутылочно-зелёный
- Бордовый
- Белый, светло-голубой
- Бежевый / песочный

⚠️ **Осторожно:**
- Чёрный — да, но не "весь чёрный сверху донизу" (старит)
- Чёрная рубашка + чёрные штаны = НЕ
- Чёрная рубашка + светлые штаны = ОК

❌ **Не идёт:**
- Коричневый, горчичный, болотный (с его оттенком кожи уходит в "уставший")
