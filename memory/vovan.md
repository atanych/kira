Контекст 2026-06-16: av.by скан 34 объявления медиана $43.7k, разброс $35.6k–$56.8k. **Battery characteristics L6**: LFP CATL → медленная деградация, гарантия 8 лет / 160k км. **Правила обслуживания LFP**: заряжать до 100% раз в неделю (нужно для калибровки BMS), не сидеть на 100% днями, не спускать ниже 20%. **SoH проверка**: через UI пути запутаны (Settings → Vehicle → Battery Health официально, но на L6 могут не вытащить). Важно: SoC (текущий заряд %) ≠ SoH (износ %!) — на главном экране Discharger показывает SoC, не SoH. **Правило тест-драйв vs частник**: -$1k разницы за тест-драйв = недостаточно. 5k тест-драйвных км ≈ 15-20k обычных (каждый второй гонял в пол). Минимум **-10% от рынка** должно быть, чтобы окупить риск. Брать с частника одним владельцем + сервисной историей.

- [[2026-06-18]] [[Vovan]] Li Auto voice commands — базовые 3 фразы (2026-06-17). Жесты: указываешь пальцем на окно/люк → произносишь команду → камера в потолке отслеживает направление пальца. Работает с салонными окнами и панорамной крышей. **1. Wake word — 理想同学** (Lǐxiǎng tóngxué) 'Лисянь тунсюэ' = 'Дружок-Идеал' (理想=Лисян бренд, 同学=приятель/одноклассник). Тоны: Ли↗ сянь↓↗ тун↗ сюэ↗ — все четыре слога вверх, как четыре вопроса подряд. **2. 打开这个** (Dǎkāi zhège) 'Дакай джэгэ' = 'Открой это'. Тоны: Да↓↗ КАЙ→ ДЖЭ↘ гэ· — 'Да' качни вниз-вверх, 'КАЙ' держи высоко, 'ДЖЭ' резко вниз, 'гэ' проглоти. **3. 关闭这个** (Guānbì zhège) 'Гуаньби джэгэ' = 'Закрой это'. Тоны: ГУАНЬ→ БИ↘ ДЖЭ↘ гэ· — 'ГУАНЬ' ровно высоко, 'БИ' роняй, 'ДЖЭ' роняй, 'гэ' глотаешь. Можно не ждать ответа ассистента, перебивать его, цеплять команды цепочкой ('开空调, 关天窗, 播放音乐'). **Для РБ**: штатная прошивка понимает только китайский. **LiNeo** — китайская кастом-прошивка для серых рынков русифицирует UI + ASR (voice по-русски) + сохраняет OTA от Li Auto. Спецы в РБ делают за ~1 час. Источники: chinamobil.ru, evmaster.by, liauto.ph.
