## Corrections / lessons

- **🚨 Не утверждать "на фото нет вспышки" если автор знает свой workflow.** Дневной fill flash на компактах (G7 X, Sony RX100) очень деликатный, легко принять за естественный свет (особенно если солнце сзади и кажется источником освещения лица). Правильно было спросить: *"ты с вспышкой это снимала или без?"* — а не заявлять диагноз по картинке. Ksusha сразу указала на ошибку, извинилась и переключилась на правильный workflow.

- **Personal Tasks closing — show what closed.** Vovan закрыл две задачи через войс, я просто подтвердила обеими галочками + 1 строкой респекта. Хорошо.

- **PDF vs HTML vs PNG для отчётов** — закрепить в [[output-style]]: для длинных отчётов с графиками — **PNG mobile-friendly** (одна простыня сверху вниз), не PDF (A4-разрывы) и не HTML (если без файла-аттача в telegram-чат не открывается просто так из messenger).

- **Chart.js: canvas reflow тупит при responsive:true в headless render.** Фикс — фиксированный canvas + responsive:false + явные barPercentage/categoryPercentage. Записала в SKILL.md market-scan.
