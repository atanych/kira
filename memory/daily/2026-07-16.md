Совет: лучше письменный запрос через nalog.gov.by → «Обратная связь» — ответ с печатью, юридическая сила. **Стратегия по режимам:** разовые проекты + мелкое сопровождение → НПД; много B2B-подписок → ИП общая система 20% с прибыли; обороты сотни тысяч BYN/год → ООО на УСН 6% (солиднее для клиентов).
