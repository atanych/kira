- **🚨 WB поиск без гендер-фильтра тащит женские карточки.** Для одежды Vovan всегда фильтровать "мужской" / искать с "мужские брюки/рубашка". Не доверять имени бренда — Stylish&Vivid например делает unisex карточки.

- **🚨 Эвристика по бренду ≠ данные.** Compogreen=Keter (нет, Prosperplast), "Prosperplast пластик толще" (по репутации, не по граммам). Vovan: *"откуда инфа?"*. Признавать когда это reputation-based, не fact-based. Лезть в реальные отзывы / спеки / вес.

- **Push back работает.** Vovan: "мульчер не подойдёт раз в месяц". Я: ровно наоборот, ты сам себе противоречишь. Принял. Не убирать аргумент в caveat — выкладывать сразу.

- **Голубика ≠ компост.** Голубике нужен pH 4-5 (кислая почва), компост даёт нейтральную/слабощелочную (pH 6-7). Для голубики: верховой торф (рыжий), хвойный опад, сера/лимонка. Малина/вишня — с компостом ок.
